Eurorack Synthesizer Modules

Passive Attenuators

EAGLE files: 3x Passive Attenuators Github repo

4HP, no power required

Three passive attenuators. This is perhaps the simplest possible Eurorack module: there are no active parts like op-amps and no power supply is required. At each output socket will appear the attenuated input signal (i.e. zero at fully counter-clockwise and max. level at fully clockwise).

The attenuators can be used to adjust the level of both CV and audio signals.

Dual Voltage-Controlled Switch

EAGLE files: Dual VCS Github repo

4HP, +/-12V

This module contains two separate voltage-controlled switches. Each switch is bi-directional. This means that they can be used to connect a single input to either one of two outputs or to connect either of two inputs to a single output. When the digital control voltage at the CV input is low, I/O1 is active; when CV is high, then I/O2 is active.

The voltage-controlled switch is implemented using a CMOS analog multiplexer DG409 (see datasheet) which means that it can be used to switch at frequencies well into the audible portion of the frequency spectrum.

Attack-Decay / Attack-Release Envelope Generator

EAGLE files: AD/AR EG Github repo

4HP, +/-12V

This is a versatile envelope generator which can generate envelopes from trigger or gate signals at the CV input. When the switch is set to "Gate", the envelope will be an attack-release envelope. When switched to "Trig", it generates attack-decay envelopes.

The implementation of this module draws heavily from a schematic designed by Ian Fritz. See below for more information.

Dual Linear Voltage-Controlled Amplifier

EAGLE files: Dual VCA Github repo

4HP, +/-12V

This module features two independent, high-fidelity voltage-controlled amplifiers which can be used for the amplification of CV or audio signals. The gain of each amplifier can vary between negative infinity (silence) and x2. Since the amplifier is capable of exceeding unity gain, it can be used to overdrive audio signals.

The gain knob sets the gain for each channel. Fully counter-clockwise is negative infinity (silence at the output), and fully clockwise is a gain of two. The CV input works with any voltage between -10V and 10V. The voltage appearing at the CV input will add to the current position of the gain potentiometer.

Low Pass Filter (24 dB per octave slope)

EAGLE files: Low Pass Filter Github repo

4HP, +/-12V

This is a 4-pole (24dB per octave) low-pass filter based around a reissue of the classic SSM2044 filter IC. This chip has provided the filter section for many famous poly synths. The two audio inputs are mixed and then filtered.

Both the cutoff frequency and the resonance level can be controlled via the potentiometers as well as by applying control voltage signals to the CV inputs.

See also: Electro-smith's 2144 LPF submodule

Buffered Multiples

Download the .brd file for the panel here: Buffered Multiples panel (right click, save as)

4HP, +/-12V

This module contains three buffered 1-to-3 multiples. The signal applied to an input socket will apear as a buffered signal at each of the three outputs. Each output is driven by its own buffer (i.e. the outputs are not connected to each other).

The input of the second multiple is normalled to the output of the first multiple, and likewise the third input is normalled to the output of the second multiple. This means that when only a single input is patched, the unit works as a 1-to-9 buffered multiple.

The LED indicators are bi-color LEDs and thus are red when the signal is negative and green when the signal is positive. The buffers are implemented using the classic low-noise op-amp, TL074, and the design is heavily influenced by the Mutable Instruments module Links (see below).

Digital Reverb BTDR-3

Files: coming soon

4HP, +/-12V

This is a monophonic digital reverb effects module. The "Reverb Level" control determines the amount of reverb signal which appears at the mix output. The "Wet" output contains only the reverb signal. This can be useful for creating interesting mixes e.g. by sending the Wet output to a VCA whose amplitude is controlled by a rhythmic envelope.

The reverb effect is implemented using BTDR-3 integrated circuit made by Belton/Accu-Bell. See the BTDR-3 datasheet here.

Balanced, Phantom-powered Microphone Pre-amp

Dimensions of PCB populated with all SMD: 21mm x 30mm x 8mm

Design files:

  • Schematic
  • SMD Balanced Mic Preamp Github repository

This is a pre-amp for an electret microphone which operates on phantom power and delivers a balanced output signal. It uses suface-mount devices and is therefore incredibly small and can easily fit inside the enclosure of a Shure SM57. The pre-amp is implemented by two transistors arranged as a Class-A op-amp.

The schematic I used is from Rod Elliott of Elliott Sound Products. According to Rod, "open loop frequency response is within 1dB from 2Hz to just under 30kHz". The pre-amp is perfect for driving almost any electret microphone as well as any other microphone that uses phantom power.

Eurorack Busboard

Dimensions of bare PCB: 355mm x 50mm

KiCad files: Eurorack Busboard Github repo

This is the first PCB I ever designed and it is a design that has seen much use. A busboard for a Eurorack synthesizer system that includes an optional 5v power supply on the busboard. The busboard has nineteen IDC headers which means it has room to power nineteen Eurorack modules.

Connections from the power supply can be made either using quick connect terminals or using JST-XH (or similar) connectors.

Stay tuned for improvements in future versions!

Power-On Delay aka "Soft Start" Timer

Dimensions of bare PCB: 50mm x 28mm
Formula for delay time: T (sec) = 1.1 * R1 * C1
Values shown: T = 1.1 * 47,000 Î© * 0.000047 F = 2.42 seconds

EAGLE files: Soft Start Github repo

It is very common for synthesizer modules to make heavy use of bypass capacitors - large electrolytic capacitors are found between almost every power rail and ground, and there are always smaller bypass caps close to the power pins of every IC. In general this is a good practice to smooth out ripples in the DC and ensure a more consistent power supply to your op-amps and other ICs. However, this means that when a synthesizer is powered on there can be a very large inrush current which can be overwhelming even for a robust power supply.

For most analog synthesizer modules, temporary power supply lags due to large inrush current do not present a problem. Some digital synth modules however, especially those implemented with microcontrollers and/or microprocessors, will fail if their required current and voltage cannot be supplied immediately when powered (i.e. they are not tolerant to "brownout" conditions). Some microcontrollers account for this with a "brownout reset" which puts the device into a reset state to ensure proper startup when power returns. Not all microcontrollers and consequently not all synthesizer modules are able to properly account for "brownout" conditions which can result in modules not functioning properly or at all. Further, relying on a microcontroller to detect and properly handle "brownout" is a tenuous solution and not one that a musician can rely upon for a performance.

Therefore the best solution to the problem of power supply brownout due to inrush current is one implemented in hardware. If modules were powered on one-by-one, or in groups, then the large inrush current could be prevented altogether. That is exactly what this "Power-on Delay" circuit (also known as a "soft start") does. It is placed between the busboard and a module and allows for a time delay before the module is powered on.

The circuit is implemented using a 555 timer and a 12V relay. The delay time is set by the value of the resistor R2 and the capacitor C1. The values shown in the schematic result in a delay of about 2.42 seconds. I use one of these for my 4MS Stereo Triggered Sampler which does not function in my system without it.
